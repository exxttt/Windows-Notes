# 📦 AmCache

## Overview

AmCache is a Windows artifact that stores information related to executed applications.

It is mainly used by Windows to improve application compatibility.

---

## 📁 Location

Typical location:
C:\Windows\AppCompat\Programs\Amcache.hve

---

## 🧩 Information Available

AmCache may contain:

- Application paths
- File metadata
- File hashes
- First execution information
- Program details

---

## 🔎 Why It Matters

AmCache can provide information about:

- Software present on a system
- Application history
- File metadata
