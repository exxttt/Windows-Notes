# 📜 Windows Event Logs

## Overview

Windows Event Logs record system activity and events generated by Windows components.

---

## 📁 Location
C:\Windows\System32\winevt\Logs

---

## 🧩 Important Logs

### Security

Contains security-related events.

Example:

- Logons
- Account changes
- Security events


### System

Contains:

- Drivers
- Services
- System errors


### Application

Contains:

- Application events
- Software errors

---

## 🔎 Why It Matters

Event Logs help understand:

- System activity
- Errors
- User actions
- Service changes

---
