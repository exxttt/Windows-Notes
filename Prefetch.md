# 🔍 Windows Prefetch

## Overview

Windows Prefetch is a Windows feature designed to improve application startup performance.

It stores information about programs that have been executed and helps Windows optimize loading times.

---

## 📁 Location C:\Windows\Prefetch or Win + R and type Prefetch 

---

## 🧩 Information Available

Prefetch files may contain:

- Application name
- Execution timestamps
- Run count
- Referenced files
- Loaded DLLs

---

## 🔎 Why It Matters

Prefetch can help understand:

- Which applications were executed
- When programs were launched
- System activity history
