# 🔌 USB History

## Overview

Windows stores information about connected USB devices.

This information can help understand device connection history.

---

## 📁 Common Locations

Registry:
HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Enum\USB

and:
HKEY_LOCAL_MACHINE\SYSTEM\MountedDevices

---

## 🧩 Information Available

USB artifacts may contain:

- Device name
- Manufacturer
- Serial number
- Connection information

---

## 🔎 Why It Matters

USB history can help identify:

- Previously connected devices
- External storage usage
- Hardware information
