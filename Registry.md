# 🗂️ Windows Registry

## Overview

The Windows Registry is a database that stores configuration settings and system information.

It contains information about Windows, users, applications and hardware.

---

## 📁 Important Locations

Common registry files:
C:\Windows\System32\Config

User registry:
NTUSER.DAT

---

## 🔎 Interesting Areas

Examples:

- User activity
- Installed applications
- System configuration
- Startup information
- Device history

---

## 🧩 Common Registry Hives

| Hive | Purpose |
|---|---|
| SYSTEM | Hardware and system settings |
| SOFTWARE | Installed software information |
| SAM | User accounts |
| SECURITY | Security settings |
| NTUSER.DAT | User-specific data |
